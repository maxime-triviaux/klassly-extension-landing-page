Guide d'Installation

Suivez ces étapes simples pour installer l'extension dans Chrome

1

📦 Télécharger le fichier ZIP

Cliquez sur le bouton de téléchargement ci-dessus pour obtenir le fichier klassly-image-downloader.zip depuis GitHub.

✅ Le fichier sera téléchargé dans votre dossier "Téléchargements"
2

📂 Décompresser le fichier ZIP

Localisez le fichier téléchargé et décompressez-le :

Sur Windows : Clic droit → "Extraire tout"

Sur Mac : Double-clic sur le fichier ZIP

Sur Linux : Clic droit → "Extraire ici"

⚠️ Gardez le dossier décompressé dans un endroit permanent (ne le supprimez pas après installation)
3

Ouvrir Chrome et accéder aux extensions

Ouvrez Google Chrome et naviguez vers la page des extensions :

chrome://extensions/

Ou utilisez le menu : ⋮ Menu → Plus d'outils → Extensions

4

🔧 Activer le mode développeur

Dans la page des extensions, activez le "Mode développeur" en cliquant sur l'interrupteur en haut à droite de la page.

💡 Le mode développeur est nécessaire pour installer des extensions non publiées sur le Chrome Web Store
5

📁 Charger l'extension

Cliquez sur le bouton "Charger l'extension non empaquetée" qui est maintenant visible.

Sélectionnez le dossier décompressé (pas le fichier ZIP) contenant l'extension.

🎉 L'extension est maintenant installée ! Vous devriez voir l'icône de Klassly Image Downloader dans votre barre d'outils Chrome.
6

✨ Utiliser l'extension

Rendez-vous sur klass.ly, ouvrez un post avec les photos en miniature et ouvrez l'extension. Lancer le scan de la page et vous verrez apparaître les boutons de téléchargement sur les images.

Cliquez dessus pour télécharger l'image !

📁 Les images seront sauvegardées dans votre dossier "Téléchargements"

🔧 Problèmes courants

L'extension n'apparaît pas :

Vérifiez que le mode développeur est bien activé et rechargez la page des extensions.

Le bouton ne s'affiche pas sur le site :

Assurez-vous d'être sur klass.ly et rechargez la page.

Erreur lors du chargement :

Vérifiez que vous sélectionnez bien le dossier décompressé, pas le fichier ZIP.
